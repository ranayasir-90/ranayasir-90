### 🔐 [EnDecLab](https://github.com/ranay/EnDecLab)
> An interactive cryptography learning platform supporting **Caesar, Rail Fence, Affine, Transposition, One-Time Pad, ROT-13** ciphers, **RSA Digital Signatures**, and **LSB Steganography**.

- 🖥️ **Stack:** Node.js · Express.js · EJS · Vanilla CSS
- ⚡ **Performance:** gzip compression · 1-year static caching · WebP images
- 🔒 **Security:** Helmet.js · CSP · HSTS headers
